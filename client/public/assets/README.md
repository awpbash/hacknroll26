# Assets Directory

## Penguin Mascot

The Quest Mode requires a penguin mascot image to be placed at:
- **Path**: `/client/public/assets/penguin-mascot.png`

### Requirements:
- Extract the penguin character from the provided reference image
- The penguin should be wearing a space suit with a star on its belly
- Recommended size: 80-100px width (height auto)
- Format: PNG with transparent background preferred
- The image will be displayed floating above the last completed checkpoint

### Fallback Behavior:
If the image is not found, the component will gracefully hide itself without causing errors.
